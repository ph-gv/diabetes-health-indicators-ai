# diabetes-health-indicators-ai
Este projeto tem como objetivo desenvolver um modelo preditivo capaz de estimar o risco de Diabetes Tipo 2 em adultos, utilizando dados clínicos, demográficos e comportamentais. A proposta está alinhada ao eixo de IA aplicada à saúde, com foco em prevenção, apoio à decisão e análise de fatores de risco.
